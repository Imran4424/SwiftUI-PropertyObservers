# SwiftUI-PropertyObservers

### State

The `@State` property wrapper is a core feature in SwiftUI that allows you to manage local, mutable data within a view. It is essential because SwiftUI views are lightweight structs (value types), which normally cannot be modified.

When the value of a `@State` property changes, SwiftUI automatically detects the modification, invalidates the view's current appearance, and recomputes its `body` to reflect the new state.
